# 3D Model Provenance (internal record)

The body models (`body_male.glb`, `body_female.glb`) are **modified derivative
works**: base meshes were normalized, re-posed, and re-exported in Blender to a
shared scale for MDAtlas. Per the project owner, the models have been altered
sufficiently that on-screen attribution is not shown in the app.

This file is kept only as an internal provenance record of the starting assets.
If licensing questions arise, confirm the base-mesh sources and terms before any
public distribution.
